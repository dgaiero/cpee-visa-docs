get_erors() → list[source]¶

Gets errors from machine.

A record of up to 20 errors is stored in the power supply’s error queue.
Errors are retrieved in first-in-first-out (FIFO) order. The first error returned is the first error that was stored.

set_output_highz()[source]¶

set_output_load(load: float)[source]¶

Select the desired output termination (i.e., the impedance of the load attached to the output of the Agilent 33210A). The specified value is used for amplitude, offset, and high/low level settings. You can set the load to any value from 1Ω to 10 kΩ.

Return type:float
Raises:MalformedParameterException

set_voltage_unit(unit: str)[source]¶

Select the units for output amplitude (does not affect offset voltage or high/low levels). The default is VPP.

Note

The output units for amplitude cannot be set to dBm if the output termination is currently set to “high impedance”.

Parameters:unit (str) – Output type of VPP or VRMS or DBM.
