class agilent_e3640a.Agilent364XA(address: str)[source]¶

Agilent Single Output Power Supply VISA interface

This class provides basic functionality for the Agilent E3640A power supply.

Note

It may work with any Agilent 364XA power supplies, however only the above has been mentioned.

get_erors() → list[source]¶

Gets errors from machine.

A record of up to 20 errors is stored in the power supply’s error queue. Errors are retrieved in first-in-first-out (FIFO) order. The first error returned is the first error that was stored.

Return type:list
